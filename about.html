Project Purpose

This site was built as a student project to show how to make a small branding guide using simple HTML and a minimal CSS framework (MVP.css).

Why a Logo Guide?

A logo guide ensures consistent branding across all forms of media. It defines how the logo should appear, which versions are acceptable, and how not to use it.

Frameworks Used

  • MVP.css
  • Custom student CSS (screen.css)

This combination gives the site a clean, professional layout without requiring complex styling.
