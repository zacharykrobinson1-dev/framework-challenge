Purpose

This website was made by a student learning basic web design and how to apply brand guidelines to a webpage. The goal was to make a simple and clean logo guide for Whole Foods Market using free no-class CSS frameworks.

What are No-Class CSS Frameworks?

No-class CSS frameworks style standard HTML elements automatically, so you don’t need to use a lot of extra classes or custom code. It’s a quick way to make a page look decent while focusing on the content.

Frameworks Used

  • MVP.css
  • Sakura.css
  • Marx.css
  • Holiday.css

Each of these frameworks adds slightly different default styles. Combining them can lead to small inconsistencies, but this project shows how they can work together for a simple student project.
